🔍 呢個 Dashboard 係咩？

追蹤港股 Market 持倉數據 — 睇邊個大戶/券商正在收貨或散貨。

🎯 點樣用嚟炒？

  1. Market% ↑ + 股價 ↑ → 大戶收貨中，跟勢追入
  2. Market% ↑ + 股價 ↓ → 大戶壓價收貨，留意見底信號
  3. Market% ↓ + 股價 ↑ → 大戶散貨中，高位減持
  4. Top5% > 80% → 高度集中，莊家股，大波動
  5. 連續增持（🔥） → 大戶持續買入，強勢信號
  6. 連續減持（❄️） → 大戶持續沽出，避開
  7. 🔺 年開突破 → 現價高過今年開市價，多頭趨勢
  8. 52週%近0% → 股價近52週低位，可能見底
  9. 52週%近100% → 股價近52週高位，小心高追

📊 量比警報 (Volume Ratio)

量比 = 今日成交量 / 平均成交量

量比信號
vr ≥ 2成交量異常放大
vr ≥ 5極端放量

常見解讀：
  • 量比 > 2 + 股價 ↑ → 放量上漲，強勢信號
  • 量比 > 2 + 股價 ↓ → 放量下跌，小心沽壓

🔍 搜尋 + 信號篩選

搜尋欄：支援股票代碼、名稱、信號類型關鍵字（如 "FVG"、"POC"、"配股"）

信號 Dropdown：主頁 toolbar 有下拉選單，可篩選特定信號類型：

  • 🎯 POC — 上破 POC（Point of Control）
  • 📅 半年POC / 3年POC — 不同時間框架
  • 📈 向上FVG — Fair Value Gap 向上
  • 🔺 年開突破 — 現價突破今年開盤價 ≥5%
  • 🆕 IPO首日突破 — 現價突破 IPO 首日高位
  • 📈 Market增持 / 📉 Market減持 — 5日持倉增減

市值按鈕：小市值 (<20億) · 中市值 (20-100億) · 大市值 (≥100億) — 三個分表獨立排序/分頁

Confluence Pattern：🚀催化 · 💎雙向 · 🦆搶跑 — 三種型態 filter
