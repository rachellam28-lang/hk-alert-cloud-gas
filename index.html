🔍 呢個 dashboard 係咩？
追蹤港股 CCASS（中央結算系統）持倉數據 — 睇邊個大戶/券商正在收貨或散貨。

📊 數據來源
• CCASS 持倉：HKEX 每日 CCASS 持股報告（持倉%、Top5/Top10 集中度、HHI）
• 股價：GAS 實時報價 + yfinance 歷史數據（今日Δ%、52週高低、PE、Beta）
• 信號：技術分析信號（POC、FVG、年開突破、IPO首日突破）
• 配股/供股：配股價、供股價突破監測

🎯 點樣用嚟炒？
1. CCASS% ↑ + 股價 ↑ → 大戶收貨中，跟勢追入
2. CCASS% ↑ + 股價 ↓ → 大戶壓價收貨，留意見底信號
3. CCASS% ↓ + 股價 ↑ → 大戶散貨中，高位減持
4. Top5% > 80% → 高度集中，莊家股，大波動
5. 連續增持（🔥） → 大戶持續買入，強勢信號
6. 連續減持（❄️） → 大戶持續沽出，避開
7. 🔺 年開突破 → 現價高過今年開市價，多頭趨勢
8. 52週%近0% → 股價近52週低位，可能見底
9. 52週%近100% → 股價近52週高位，小心高追

📌 關鍵數字解讀
• 持倉%（tp）：CCASS 總持股佔發行量%，>60%即高集中
• 5日Δ（d5）：近5日CCASS%變化，+值=收貨，-值=散貨
• 今日Δ%（chg）：今日股價變幅（yfinance）
• PE：市盈率，<10可能低估，>100可能高估
• Beta：波動性，>1比大市波動大
• HHI（drawer入面）：持倉集中度指數，>1500=高度集中

⚡ 快篩 Presets
• 信號突破：按技術信號數量排序
• 配股預警：CCASS%>30 + 有配股記錄
• 高動能：5日Δ跌5-20%（大戶低位掃貨）

🔄 更新頻率
• CCASS 數據：每日 HKEX update 後（約7am HKT）
• 股價 + 停牌：每日 9pm HKT yfinance refresh
• 信號：GAS 自動推送

🔬 數據精準度
另類數據嘅靈魂在於清洗顆粒度與精準度，及深度關聯處理。
• 8重驗證：range（持倉% 0-100）、pct consistency（daily vs holdings sum ≤5%誤差）、shares consistency（總股 vs 持倉sum ≤2%）、day-over-day jump（>30%跳變檢測）、coverage gap（<50% median → alert）、zero participant（有股數無參與者）、orphan row（daily無對應holdings）、concentration sanity（top5>total）
• NaN/Infinity sanitize：export JSON 前 math.isnan() → None，防 dashboard 白畫面
• 幽靈數據防護：fill_missing DELETE before INSERT，避免重複 scrape 疊加
• Scraper bug detection：pct 大變但 shares 不變 → 判定為 scraper error
• Corporate action 識別：pct + shares 同時大變 → 判定為公司行動（非數據錯誤）
• WAF protection：sequential scrape only，subprocess 60s hard timeout，fail rate >5% → admin alert
• 腳本：verify_dashboard.py（JSON層） + verify_data.py（DB 8重深度檢查）
