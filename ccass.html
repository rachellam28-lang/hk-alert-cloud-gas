321| 🔍 呢個 dashboard 係咩？
322| 追蹤港股 CCASS（中央結算系統）持倉數據 — 睇邊個大戶/券商正在收貨或散貨。

323| 324| 📊 數據來源
325| • CCASS 持倉：HKEX 每日 CCASS 持股報告（持倉%、Top5/Top10 集中度、HHI）
326| • 股價：GAS 實時報價 + yfinance 歷史數據（今日Δ%、52週高低、PE、Beta）
327| • 信號：技術分析信號（POC、FVG、年開突破、IPO首日突破）
328| • 配股/供股：配股價、供股價突破監測

329| 330| 🎯 點樣用嚟炒？
331| 1. CCASS% ↑ + 股價 ↑ → 大戶收貨中，跟勢追入
332| 2. CCASS% ↑ + 股價 ↓ → 大戶壓價收貨，留意見底信號
333| 3. CCASS% ↓ + 股價 ↑ → 大戶散貨中，高位減持
334| 4. Top5% > 80% → 高度集中，莊家股，大波動
335| 5. 連續增持（🔥） → 大戶持續買入，強勢信號
336| 6. 連續減持（❄️） → 大戶持續沽出，避開
337| 7. 🔺 年開突破 → 現價高過今年開市價，多頭趨勢
338| 8. 52週%近0% → 股價近52週低位，可能見底
339| 9. 52週%近100% → 股價近52週高位，小心高追

340| 341| 📌 關鍵數字解讀
342| • 持倉%（tp）：CCASS 總持股佔發行量%，>60%即高集中
343| • 5日Δ（d5）：近5日CCASS%變化，+值=收貨，-值=散貨
344| • 今日Δ%（chg）：今日股價變幅（yfinance）
345| • PE：市盈率，<10可能低估，>100可能高估
346| • Beta：波動性，>1比大市波動大
347| • HHI（drawer入面）：持倉集中度指數，>1500=高度集中

348| 349| ⚡ 快篩 Presets
350| • 信號突破：按技術信號數量排序
351| • 配股預警：CCASS%>30 + 有配股記錄
352| • 高動能：5日Δ跌5-20%（大戶低位掃貨）

353| 354| 🔄 更新頻率
355| • CCASS 數據：每日 HKEX update 後（約7am HKT）
356| • 股價 + 停牌：每日 9pm HKT yfinance refresh
357| • 信號：GAS 自動推送
